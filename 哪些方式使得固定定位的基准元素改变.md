# 哪些方式使得position:fixed定位的基准元素改变？
当元素祖先的  
1.transform属性为非none  
2.perspective属性为非none  
3.filter 属性非 none  
时，容器由视口改为该祖先

## 例子
```
css
.box {
  position: relative;
  top: 200px;
  /* transform: rotate(30deg); */
  /* perspective: 100px */
  filter: brightness(0.5);
}

.box div {
  background: black;
  width: 200px;
  height: 200px;
  position: fixed;
  top: 50px;
  left: 50px;
}

html
<div class="box">
  <div></div>
</div>
```
